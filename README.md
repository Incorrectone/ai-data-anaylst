# AI Data Analyst Project

Inspiration from Tools for Data Science Course Project. Made to learn about React.JS, LangChain, RAG.

## Tech Stack Used
1. Docker for PostgresSQL with SQLAlchemy
2. Next.JS (React) for Frontend
3. FastAPI for Backend
4. LlamaIndex or LangChain
5. LLMs for Chained Thinking (Most likely gpt-5-nano for cheapness)
6. E2B for sandboxed code running
7. Vector Database using Pinecone (maybe)

See individual folder ```README.md``` for instruction to get running.

## Docker Setup Instruction
In the project root directory:
```bash
sudo docker compose up -d # Start Services
sudo docker ps # Check Status
```

## Intention for the Project

To Learn the  basics of Web Dev and Agentic AIs
